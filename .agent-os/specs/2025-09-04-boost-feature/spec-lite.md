# Character Boost Feature - Lite Summary

Enable tactical support in combat through character boost actions that provide temporary attack or defense bonuses to allies, following Feng Shui 2 mechanics with optional Fortune die enhancement.

## Key Points
- 3-shot action to boost ally's next attack (+1/+2) or defense (+3/+5)
- Fortune die enhancement available for PC characters
- Mirrors Attack Panel UI patterns for familiar user experience
- CharacterEffect records track active boosts with real-time updates
- Integrates seamlessly with existing combat and shot counter systems
