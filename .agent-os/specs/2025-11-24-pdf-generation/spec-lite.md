# PDF Generation for Elixir Backend - Lite Summary

Implement character sheet PDF export and import functionality in the Elixir/Phoenix backend to match existing Rails behavior, enabling offline character management and integration with fillable PDF forms.

## Key Points
- Export characters as fillable PDF sheets with all stats, schticks, and weapons
- Import character data from filled PDF forms to create new character records
- Integrate ChromicPDF library for HTML-to-PDF rendering with form field preservation
